# Calculadora em Java

Meu primeiro projeto em Java: uma calculadora simples executada pelo terminal.

## Sobre o projeto

Este projeto foi criado com o objetivo de praticar os conceitos iniciais da linguagem Java, como variáveis, entrada de dados, estrutura condicional `switch` e operações matemáticas básicas.

## Funcionalidades

- Soma
- Subtração
- Multiplicação
- Divisão
- Menu interativo no terminal

## Tecnologias utilizadas

- Java
- VS Code
- Git
- GitHub

## Como executar o projeto

1. Clone este repositório:

```bash
git clone URL_DO_SEU_REPOSITORIO
````

2. Entre na pasta do projeto:

```bash
cd calculadora-java
```

3. Execute o arquivo Java:

```bash
java src/Main.java
```

## Exemplo de uso

```text
=== Calculadora ===
1 - Soma
2 - Subtração
3 - Multiplicação
4 - Divisão

Escolha uma opção:
1

Digite o primeiro número:
10

Digite o segundo número:
5

Resultado da soma: 15
```
